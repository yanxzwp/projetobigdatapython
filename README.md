Projeto Big Data – Sistema de Recomendação de Filmes e Séries

Este projeto é um sistema de recomendação de filmes e séries feito em Python usando Streamlit, pandas e scikit-learn.

Como rodar o projeto
Passo 1 – Baixar o projeto

Entre no link do GitHub: https://github.com/yanxzwp/projetobigdatapython

Clique em Code → Download ZIP

Descompacte o arquivo ZIP em alguma pasta no computador.

Passo 2 – Instalar Python

Certifique-se de que o computador tem o Python 3 instalado: https://www.python.org/downloads/

Passo 3 – Instalar as bibliotecas necessárias

Abra o Prompt de Comando (Windows) ou Terminal (macOS/Linux)

Navegue até a pasta do projeto. Exemplo:

cd C:\Users\SeuUsuario\Downloads\trabalho-bigdata


Instale as bibliotecas do projeto:

pip install -r requirements.txt

Passo 4 – Rodar o Streamlit

Ainda no Prompt de Comando ou Terminal, execute:

streamlit run app.py


O navegador vai abrir automaticamente mostrando o sistema de recomendação.

Observações

Todas as imagens usadas no projeto estão dentro da pasta imagens/.

Se algum arquivo de imagem estiver faltando, o Streamlit mostrará uma mensagem “Imagem não disponível”.
